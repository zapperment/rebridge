# ReBridge

MIDI controller setup for Launch Control XL3 + Ableton Live + Reason

## How to install

There is a script, `setup.sh`, which will copy the files in the `dist` directory to the appropriate target directories in the Ableton Live user library and the Reason remote codec and map directories.

To make this work, make a copy of `.env.example` and rename it to `.env`.

Adjust the paths in this file according to your setup.

`.env.devcontainer` is for development only.
